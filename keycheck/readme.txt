RUS:
Библиотека предназначена для проверки сгенерированного ключа по собственному алгоритму.
Файл содержит несколько функций для проверки ключа.
Функции находятся в классе "KeyCheck", расположенном в пространстве имен "kc_log".
Алгоритм использования библиотеки:
1. Использовать пространство имен (название указано выше).
2. Создать экземпляр класса (название класса указано выше).
3. Вызвать единственную доступную функцию checkKey и передать туда ключ.

Будьте внимательны при использовании данной библиотеки.

Замечания:
1. Ключ должен иметь тип строки (std::string).
